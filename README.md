# mnist-classification
使用多种方法完成MNIST分类任务

Python 3.6

Pytorch 1.0

Scikit-learn 0.21

无需下载数据直接跑，代码自动下载




## 模型

* 逻辑回归 Logistic Regression
* 多层感知机 MLP
* K近邻 KNN
* 支持向量机 SVM
* 卷积神经网络 CNN
* 循环神经网络 RNN
